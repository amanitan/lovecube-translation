言語 MOD 作成方法

言語 MOD を作成するには4種類のファイルを準備する必要があります。
まず、言語(日本語、英語)別に MOD のテンプレートファイルが用意されているので、それらファイルをダウンロードして確認してください。
JSON と シナリオファイルは UTF-8 で保存してください。

必要なファイル
1. ModFont.ttf
	MOD で翻訳したテキストを表示するためのフォントです。
	MOD の時このフォントで UI やシナリオが描画されます。
	TTFファイルまたはOTFファイルが使用できます (OTFファイルの場合も拡張子はttfにして配置してください)。
	翻訳したテキストの字形が含まれているフォントで再配布可能なものを準備してください。
	フォントを再配布する際にライセンス上に問題はないのかを確認してください。
2. ui_text.json
	UI のテキストです。
	グループごとに分類され、各要素の左側が ID で、右側が表示されるテキストです。
	右側のテキストを翻訳してください。
	inputmemo の %02d の部分には数値が入ります。
	basic_preview_text はコンフィグでメッセージウィンドウに表示されるテキストのプレビューに使用されます。
	配列で、3つのテキストを翻訳してください。
3. chapter_title.json
	各シナリオのチャプター名です。
	右側の各章のタイトルを翻訳してください。
	チャプター名は、セーブデータで確認できます。
4. eyecatch.json
	アイキャッチのテキストです。
	各画像のテキストを翻訳してください。
	アイキャッチのテキストは、ゲーム本編で確認できます。
5. シナリオファイル(.msファイル)
	シナリオファイルです。
	ゲーム中のメッセージで表示される、実際のシナリオです。
	後述の注意点を見て、翻訳してください。
	ファイル名はテンプレートから変更しないでください。
	また表示行数や位置が違うとゲーム中の演出と合わなくなるので注意してください。
	文章が長すぎると表示しきれない場合もあります。


単語の途中改行について

英単語は途中改行されないようになっていますが、英語とは異なる文字を持つ場合、途中改行されてしまいます。
その場合、ui_text.json の message の wordwrap_latin に文字を追加してください。
テンプレートに書かれている "abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789'-" は、記載例で、これはこのまま残す必要はありません。
追加したい文字のみ追加してください。


シナリオファイル翻訳上の注意点

@ で始まる行はキャラ名を表しています。@の後にキャラクター名を入れます。キャラクター名の途中に記号やスペースが含まれている場合は、""で囲う必要があります。
// で始まる行はコメント行で、無視されます。訳す必要はありません。
# で始まる行はラベルです。内部で使用されますが、訳す必要はありません。
単言語モードでは240文字ぐらい表示できますが、二言語表示モードをサポートするには170文字までに制限する必要があります。
三行とその以上の同時音声の場合、行数を削ることもあります。

特殊記号
シナリオファイルでは、いくつかの記号は特殊な意味を持つため、文章中で使用する場合は、エスケープする必要があります。
エスケープは、記号の前に\(バックスラッシュ)をいれることで行います。
文章中で特別扱いされる記号は以下の物です。
[]|《》{}>:=
