lovecube の MOD 機能は、言語を追加するための機能です。
「マイドキュメント/NEKO WORK H/LOVE CUBE/MOD/」以下に公開されている言語MODをコピーすることで、その言語に置き換えが可能です。
各ファイルは MOD フォルダ直下に置きます。

配置場所例 : 
マイドキュメント/NEKO WORK H/LOVE CUBE/MOD/ModFont.ttf
マイドキュメント/NEKO WORK H/LOVE CUBE/MOD/ui_text.json
マイドキュメント/NEKO WORK H/LOVE CUBE/MOD/chapter_title.json
マイドキュメント/NEKO WORK H/LOVE CUBE/MOD/シナリオファイル(*.ms)


言語MODには、以下の4種類のファイルが含まれます。

1. ModFont.ttf : MOD で翻訳したテキストを表示するためのフォントです。
2. ui_text.json : UI のテキストを翻訳したファイルです。
3. chapter_title.json : 各シナリオのチャプター名ほ翻訳したファイルです。
4. シナリオファイル : シナリオを翻訳したファイルです。

1.
フォントは翻訳した文字が表示できるフォントを用意します。
このファイルがない場合は英語と同じフォントが使用されます。

2.
UI のテキストを翻訳したものを用意します。
英語や日本語で対応する箇所を確認できます。
対象のファイルがなかったり、正常に読み込めない場合は英語で表示されます。

3.
チャプター名を翻訳したものを用意します。
対象のファイルがなかったり、正常に読み込めない場合は英語で表示されます。

4.
各シナリオファイルを翻訳したものを用意します。
対象のファイルがない場合は英語で表示されます。
